SynchronizedжҖ»з»“

жүҖжңүзҡ„е№¶еҸ‘жЁЎејҸеңЁи§ЈеҶізәҝзЁӢе®үе…Ёй—®йўҳж—¶пјҢйҮҮз”Ёзҡ„ж–№жЎҲйғҪжҳҜеәҸеҲ—еҢ–и®ҝй—®дёҙз•Ңиө„жәҗгҖӮеҚіеҗҢдёҖж—¶еҲ»еҸӘиғҪжңүдёҖдёӘзәҝзЁӢи®ҝй—®дёҙз•Ңиө„жәҗпјҢд№ҹз§°дҪңеҗҢжӯҘдә’ж–Ҙи®ҝй—®гҖӮ JavaжҸҗдҫӣsynchronizedе’ҢLockдёӨз§Қж–№ејҸжқҘе®һзҺ°еҗҢжӯҘдә’ж–Ҙи®ҝй—®гҖӮ

synchronizedеҶ…зҪ®й”ҒжҳҜдёҖз§ҚеҜ№иұЎй”ҒпјҢй”Ғзҡ„жҳҜеҜ№иұЎиҖҢйқһеј•з”ЁпјҢдҪңз”ЁзІ’еәҰжҳҜеҜ№иұЎпјҢеҸҜз”ЁжқҘе®һзҺ°еҜ№дёҙз•Ңиө„жәҗзҡ„еҗҢжӯҘдә’ж–Ҙи®ҝй—®пјҢжҳҜеҸҜйҮҚе…Ҙзҡ„пјҢжҳҜJavaдёӯи§ЈеҶіе№¶еҸ‘й—®йўҳзҡ„дёҖз§ҚжңҖеёёз”ЁжңҖз®ҖеҚ•зҡ„ж–№жі•пјҢе®ғеҸҜд»ҘзЎ®дҝқзәҝзЁӢдә’ж–Ҙзҡ„и®ҝй—®еҗҢжӯҘд»Јз ҒпјҢдҝқиҜҒж–№жі•жҲ–иҖ…д»Јз Ғеқ—еңЁиҝҗиЎҢж—¶пјҢеҗҢдёҖж—¶еҲ»еҸӘжңүдёҖдёӘж–№жі•еҸҜд»Ҙиҝӣе…ҘеҲ°дёҙз•ҢеҢәпјҢеҗҢж—¶е®ғиҝҳеҸҜд»ҘдҝқиҜҒе…ұдә«еҸҳйҮҸзҡ„еҶ…еӯҳеҸҜи§ҒжҖ§гҖӮ

еҺҹзҗҶ

иҷҡжӢҹжңәж”ҜжҢҒж–№жі•зә§зҡ„еҗҢжӯҘе’Ңж–№жі•еҶ…йғЁдёҖж®өжҢҮд»ӨеәҸеҲ—зҡ„еҗҢжӯҘпјҢдёӨз§ҚеҗҢжӯҘйғҪдҪҝз”Ёз®ЎзЁӢMonitorжқҘж”ҜжҢҒзҡ„гҖӮ

ж–№жі•зә§зҡ„еҗҢжӯҘжҳҜйҡҗејҸзҡ„пјҢж— йЎ»йҖҡиҝҮеӯ—иҠӮз ҒжҢҮд»ӨжқҘжҺ§еҲ¶пјҢе®ғе®һзҺ°еңЁж–№жі•и°ғз”Ёе’Ңиҝ”еӣһж“ҚдҪңд№ӢдёӯгҖӮиҷҡжӢҹжңәеҸҜд»Ҙд»Һж–№жі•еёёйҮҸжұ зҡ„ж–№жі•з»“жһ„дёӯзҡ„ACC_SYNCHRONIZEDи®ҝй—®ж Үеҝ—еҫ—зҹҘж–№жі•жҳҜеҗҰеЈ°жҳҺдёәеҗҢжӯҘж–№жі•гҖӮеҪ“ж–№жі•и°ғз”Ёж—¶пјҢи°ғз”Ёзҡ„жҢҮд»Өе°ҶдјҡжЈҖжҹҘж–№жі•зҡ„ACC_SYNCHRONIZEDи®ҝй—®ж Үеҝ—жҳҜеҗҰиў«и®ҫзҪ®пјҢиӢҘиў«и®ҫзҪ®пјҢжү§иЎҢзәҝзЁӢе°ұиҰҒжұӮе…ҲжҲҗеҠҹжҢҒжңүз®ЎзЁӢMonitorпјҢз„¶еҗҺжүҚиғҪжү§иЎҢж–№жі•пјҢжңҖеҗҺеҪ“ж–№жі•жү§иЎҢе®ҢжҲҗпјҢж— и®әжҳҜеҗҰжӯЈеёёе®ҢжҲҗйғҪйҮҠж”ҫз®ЎзЁӢMonitorпјҢж–№жі•жү§иЎҢжңҹй—ҙпјҢжү§иЎҢзәҝзЁӢжҢҒжңүдәҶз®ЎзЁӢMonitorпјҢе…¶д»–д»»дҪ•зәҝзЁӢйғҪж— жі•еҶҚиҺ·еҫ—еҗҢдёҖдёӘз®ЎзЁӢMonitorгҖӮиӢҘеҗҢжӯҘж–№жі•жү§иЎҢжңҹй—ҙжҠӣеҮәејӮеёёпјҢдё”ж–№жі•еҶ…йғЁж— жі•еӨ„зҗҶејӮеёёпјҢеҗҢжӯҘж–№жі•жүҖжҢҒжңүзҡ„з®ЎзЁӢMonitorе°ҶеңЁејӮеёёжҠӣеҲ°еҗҢжӯҘж–№жі•еӨ–ж—¶иҮӘеҠЁйҮҠж”ҫгҖӮ

Monitorзӣ‘и§ҶеҷЁй”Ғзҡ„е®һзҺ°дҫқиө–еә•еұӮж“ҚдҪңзі»з»ҹзҡ„Mutex lockдә’ж–Ҙй”Ғе®һзҺ°пјҢжҳҜдёҖдёӘйҮҚйҮҸзә§й”ҒжҖ§иғҪиҫғдҪҺгҖӮJava1.5д№ӢеҗҺзүҲжң¬еҒҡдәҶйҮҚеӨ§зҡ„дјҳеҢ–пјҢеҰӮй”ҒзІ—еҢ–гҖҒй”Ғж¶ҲйҷӨгҖҒиҪ»йҮҸзә§й”ҒгҖҒеҒҸеҗ‘й”ҒгҖҒйҖӮеә”жҖ§иҮӘж—ӢзӯүжҠҖжңҜжқҘеҮҸе°‘й”Ғж“ҚдҪңзҡ„ејҖй”ҖгҖӮ

еҗҢжӯҘдёҖж®өжҢҮд»ӨйӣҶеәҸеҲ—йҖҡеёёжҳҜйҖҡиҝҮsynchronizedиҜӯеҸҘеқ—жқҘе®ҢжҲҗпјҢиҷҡжӢҹжңәзҡ„жҢҮд»ӨйӣҶдҪҝз”Ёmonitorenterе’ҢmonitorexitдёӨжқЎжҢҮд»ӨжқҘж”ҜжҢҒsynchronizedе…ій”®еӯ—зҡ„иҜӯд№үпјҢжӯЈзЎ®е®һзҺ°synchronizedе…ій”®еӯ—йңҖиҰҒJavacзј–иҜ‘еҷЁе’ҢJavaиҷҡжӢҹжңәе…ұеҗҢеҚҸдҪңж”ҜжҢҒпјҢзј–иҜ‘еҷЁеҝ…йЎ»дҝқиҜҒж–№жі•йҖҡиҝҮд»»дҪ•ж–№ејҸе®ҢжҲҗпјҢж–№жі•дёӯи°ғз”ЁиҝҮзҡ„жҜҸжқЎmonitorenterжҢҮд»ӨйғҪеҝ…йЎ»жү§иЎҢе…¶еҜ№еә”зҡ„monitorexitжҢҮд»ӨпјҢж— и®әиҜҘж–№жі•жҳҜеҗҰжӯЈеёёз»“жқҹгҖӮ

дёәдәҶдҝқиҜҒж–№жі•ејӮеёёе®ҢжҲҗж—¶monitorenterе’ҢmonitorexitжҢҮд»ӨиғҪжӯЈзЎ®й…ҚеҜ№жү§иЎҢпјҢзј–иҜ‘еҷЁдјҡиҮӘеҠЁдә§з”ҹдёҖдёӘејӮеёёеӨ„зҗҶеҷЁпјҢдё”еЈ°жҳҺеҸҜеӨ„зҗҶзҡ„жүҖжңүејӮеёёпјҢжқҘжү§иЎҢmonitorexitжҢҮд»ӨгҖӮ

monitorenterе’ҢmonitorexitиҝҷдёӨдёӘеӯ—иҠӮз ҒйғҪйңҖиҰҒдёҖдёӘreferenceзұ»еһӢзҡ„еҸӮж•°жқҘжҢҮжҳҺиҰҒй”Ғе®ҡе’Ңи§Јй”Ғзҡ„еҜ№иұЎгҖӮиӢҘsynchronizedжҳҺзЎ®жҢҮе®ҡдәҶеҜ№иұЎеҸӮж•°пјҢйӮЈе°ұжҳҜиҝҷдёӘеҜ№иұЎзҡ„referenceпјҢиӢҘжІЎжңүжҳҺзЎ®жҢҮе®ҡпјҢе°ұж №жҚ®synchronizedдҝ®йҘ°зҡ„жҳҜе®һдҫӢж–№жі•иҝҳжҳҜзұ»ж–№жі•пјҢеҲҷеҸ–еҜ№еә”зҡ„еҜ№иұЎе®һдҫӢжҲ–ClassеҜ№иұЎжқҘдҪңдёәй”ҒеҜ№иұЎгҖӮ

monitorexitжҢҮд»ӨеҮәзҺ°дәҶдёӨж¬ЎпјҢз¬¬1ж¬ЎдёәеҗҢжӯҘжӯЈеёёйҖҖеҮәйҮҠж”ҫй”Ғпјӣз¬¬2ж¬ЎдёәеҸ‘з”ҹејӮжӯҘйҖҖеҮәйҮҠж”ҫй”Ғпјӣwait/notifyзӯүж–№жі•д№ҹдҫқиө–дәҺmonitorеҜ№иұЎпјҢиҝҷе°ұжҳҜдёәд»Җд№ҲеҸӘжңүеңЁеҗҢжӯҘзҡ„еқ—жҲ–иҖ…ж–№жі•дёӯжүҚиғҪи°ғз”Ёwait/notifyзӯүж–№жі•пјҢеҗҰеҲҷдјҡжҠӣеҮәjava.lang.IllegalMonitorStateExceptionејӮеёёгҖӮ

иҷҡжӢҹжңәи§„иҢғиҰҒжұӮпјҢжү§иЎҢmonitorenterжҢҮд»Өж—¶пјҢйҰ–е…Ҳе°қиҜ•иҺ·еҸ–еҜ№иұЎзҡ„й”ҒпјҢиӢҘеҜ№иұЎжІЎжңүиў«й”Ғе®ҡжҲ–еҪ“еүҚзәҝзЁӢе·Із»ҸжӢҘжңүиҝҷдёӘеҜ№иұЎзҡ„й”ҒпјҢе°Ҷй”Ғзҡ„и®Ўж•°еҷЁеҠ дёҖпјҢжү§иЎҢmonitorexitжҢҮд»Өж—¶е°Ҷй”Ғи®Ўж•°еҷЁеҮҸдёҖпјҢеҪ“и®Ўж•°еҷЁдёәйӣ¶ж—¶й”Ғиў«йҮҠж”ҫгҖӮиӢҘиҺ·еҸ–еҜ№иұЎй”ҒеӨұиҙҘпјҢеҪ“еүҚзәҝзЁӢйҳ»еЎһзӯүеҫ…пјҢзӣҙеҲ°еҜ№иұЎй”Ғиў«еҸҰдёҖдёӘзәҝзЁӢйҮҠж”ҫгҖӮ

synchronizedеҗҢжӯҘеқ—еҜ№еҗҢдёҖзәҝзЁӢжҳҜеҸҜйҮҚе…Ҙзҡ„пјҢдёҚдјҡе°ҶиҮӘе·ұжӯ»й”ҒпјҢдҪҶжҳҜжҳҜдёҚеҸҜдёӯж–ӯзҡ„гҖӮеҗҢжӯҘеқ—еңЁе·Іиҝӣе…Ҙзҡ„зәҝзЁӢжү§иЎҢе®ҢжҲҗд№ӢеүҚпјҢдјҡйҳ»еЎһеҗҺйқўе…¶д»–зәҝзЁӢзҡ„иҝӣе…ҘгҖӮJavaзәҝзЁӢжҳҜжҳ е°„еҲ°ж“ҚдҪңзі»з»ҹеҺҹз”ҹзәҝзЁӢдёҠзҡ„пјҢйҳ»еЎһжҲ–е”ӨйҶ’зәҝзЁӢйғҪйңҖиҰҒж“ҚдҪңзі»з»ҹеё®еҝҷпјҢйңҖиҰҒд»Һз”ЁжҲ·зҠ¶жҖҒиҪ¬жҚўеҲ°ж ёеҝғжҖҒдёӯпјҢеӣ жӯӨиҪ¬жҖҒиҪ¬жҚўйңҖиҰҒиҖ—иҙ№еҫҲеӨҡеӨ„зҗҶеҷЁж—¶й—ҙгҖӮ

еҜ№иұЎеӨҙMarkWordй”Ғж ҮиҜҶдҪҚдёә10пјҢе…¶дёӯжҢҮй’ҲжҢҮеҗ‘зҡ„жҳҜMonitorеҜ№иұЎзҡ„иө·е§Ӣең°еқҖпјҢMonitorжҳҜз”ұObjectMonitorе®һзҺ°зҡ„пјҢе…¶дҪҚдәҺHotSpotиҷҡжӢҹжңәжәҗз ҒObjectMonitor.hppж–Үд»¶гҖӮ

waitдёҺnotify

waitе’ҢnotifyйғҪжҳҜObjectзҡ„жҲҗе‘ҳеҮҪж•°пјҢеӣ дёәsynchronizedе…ій”®еӯ—еҸҜд»ҘеҠ еңЁд»»дҪ•еҜ№иұЎзҡ„жҲҗе‘ҳеҮҪж•°дёҠпјҢд»»дҪ•еҜ№иұЎйғҪеҸҜд»ҘжҲҗдёәй”ҒпјҢжүҖд»Ҙwaitе’ҢnotifyеҸӘиғҪж”ҫеңЁObjectйҮҢйқўгҖӮ

waitеҶ…йғЁдјҡе…ҲйҮҠж”ҫй”ҒпјҢз„¶еҗҺиҝӣе…Ҙйҳ»еЎһзҠ¶жҖҒпјҢеҪ“иў«еҸҰдёҖдёӘзәҝзЁӢnotifyеҗҺпјҢйҮҚж–°жӢҝй”ҒпјҢеңЁжү§иЎҢеҗҺз»ӯйҖ»иҫ‘гҖӮ

еҜ№дәҺз”ҹдә§иҖ…ж¶Ҳиҙ№иҖ…жЁЎеһӢжқҘиҜҙпјҢз”ұдәҺwaitе’ҢnotifyжүҖдҪңз”ЁеҜ№иұЎе’ҢsynchronizedдҪңз”Ёзҡ„еҜ№иұЎжҳҜеҗҢдёҖдёӘпјҢеҸӘиғҪжҳҜдёҖдёӘеҜ№иұЎпјҢж— жі•еҢәеҲҶйҳҹеҲ—з©әжҲ–йҳҹеҲ—ж»ЎдёӨдёӘжқЎд»¶гҖӮ

дҪҝз”ЁеңәжҷҜ

JavaдёӯдҪҝз”ЁsynchronizedеҸҜдҪҝз”ЁеңЁд»Јз Ғеқ—е’Ңж–№жі•дёӯпјҢsynchronizedз”Ёзҡ„дҪҚзҪ®дёҚеҗҢй”Ғеҫ—еҜ№иұЎе°ұдёҚеҗҢгҖӮ

еҲҶзұ» иў«й”ҒеҜ№иұЎ е®һдҫӢ
е®һдҫӢж–№жі• зұ»еҫ—е®һдҫӢеҜ№иұЎ public synchronized void sync1() {}
йқҷжҖҒж–№жі• зұ»еҜ№иұЎ public static synchronized void sync2() {}
е®һдҫӢеҜ№иұЎ зұ»еҫ—е®һдҫӢеҜ№иұЎ synchronized (this) {}
classеҜ№иұЎ зұ»еҜ№иұЎ synchronized (SynchronizedDemo.class) {}
д»»ж„Ҹе®һдҫӢеҜ№иұЎObject е®һдҫӢеҜ№иұЎObject String lock = ""; synchronized (lock) {}

еҪ“дёҖдёӘзәҝзЁӢи®ҝй—®жҹҗеҜ№иұЎзҡ„synchronizedж–№жі•жҲ–иҖ…synchronizedд»Јз Ғеқ—ж—¶пјҢе…¶д»–зәҝзЁӢеҜ№иҜҘеҜ№иұЎзҡ„иҜҘsynchronizedж–№жі•жҲ–иҖ…synchronizedд»Јз Ғеқ—зҡ„и®ҝй—®е°Ҷиў«йҳ»еЎһпјӣе…¶д»–зәҝзЁӢд»ҚеҸҜд»Ҙи®ҝй—®иҜҘеҜ№иұЎзҡ„йқһеҗҢжӯҘд»Јз Ғеқ—пјӣе…¶д»–зәҝзЁӢеҜ№иҜҘеҜ№иұЎзҡ„е…¶д»–зҡ„synchronizedж–№жі•жҲ–иҖ…synchronizedд»Јз Ғеқ—зҡ„и®ҝй—®е°Ҷиў«йҳ»еЎһгҖӮ

еҗҢжӯҘд»Јз Ғеқ—

й”Ғзҡ„еҜ№иұЎжҳҜжӢ¬еҸ·йҮҢйқўзҡ„thisеҜ№иұЎ

жңҖз»Ҳжү“еҚ°з»“жһңеҰӮдёӢпјҢеҸҜд»ҘеҫҲжҳҺжҳҫзҡ„зңӢеҮәrunж–№жі•йҮҢзҡ„д»Јз ҒжҳҜйЎәеәҸжү§иЎҢгҖӮ

еҰӮдёӢеҶҷжі•иҝҗиЎҢз»“жһңдёҖж ·пјҢйҷӨдәҶдҪҝз”ЁthisеҜ№иұЎдҪңдёәй”ҒпјҢд№ҹеҸҜд»Ҙе®ҡд№үдёҖдёӘдё“й—Ёзҡ„й”ҒеҜ№иұЎгҖӮ

еҸҜд»Ҙе®ҡд№үеӨҡдёӘй”ҒеҜ№иұЎ

жңҖз»Ҳжү“еҚ°з»“жһңеҰӮдёӢпјҢеҸҜд»ҘзңӢеҮәrunж–№жі•йҮҢзҡ„д»Јз ҒжҳҜж—¶иў«жү§иЎҢзҡ„гҖӮеҪ“thread1йҮҠж”ҫжҺүlock1зҡ„й”Ғж—¶пјҢthread2еҚіеҸҜиҺ·еҫ—lock1зҡ„й”ҒпјҢжүҖд»Ҙthread2зҡ„lock1дёӯзҡ„еҗҢжӯҘд»Јз Ғеқ—еҸҜд»ҘдёҺthread1зҡ„lock2дёӯзҡ„еҗҢжӯҘд»Јз Ғеқ—е№¶иЎҢжү§иЎҢгҖӮ

ж–№жі•й”Ғ

й”ҒеҜ№иұЎжҳҜеҪ“еүҚе®һдҫӢеҜ№иұЎ

жңҖз»Ҳжү“еҚ°з»“жһңеҰӮдёӢ

зұ»й”Ғ

й”ҒеҜ№иұЎжҳҜеҪ“еүҚзұ»еҜ№иұЎпјҢJavaзұ»еҸҜиғҪжңүеӨҡдёӘеҜ№иұЎпјҢдҪҶеҸӘжңүдёҖдёӘClassеҜ№иұЎпјҢиҖҢзұ»й”Ғзҡ„й”ҒеҜ№иұЎе°ұжҳҜиҜҘзұ»зҡ„ClassеҜ№иұЎпјҢзұ»й”ҒжңүдёӨз§ҚеҪўејҸпјҡе°ҶsynchronizedеҠ еңЁstaticж–№жі•дёҠе’Ңsynchronized(*.class)д»Јз Ғеқ—гҖӮsynchronizedеҠ еңЁstaticж–№жі•дёҠпјҡ

жңҖз»Ҳжү“еҚ°з»“жһңеҰӮдёӢ

synchronized(*.class)д»Јз Ғеқ—пјҡ

жңҖз»Ҳжү“еҚ°з»“жһңеҰӮдёӢ

еӨҡзәҝзЁӢи®ҝй—®synchronizedеҗҢжӯҘж–№жі•еёёи§Ғзҡ„7з§Қжғ…еҶөпјҡ

  • дёӨдёӘзәҝзЁӢеҗҢж—¶и®ҝй—®дёҖдёӘеҜ№иұЎзҡ„еҗҢжӯҘж–№жі•пјҡйҳ»еЎһ
  • дёӨдёӘзәҝзЁӢи®ҝй—®зҡ„жҳҜдёӨдёӘеҜ№иұЎзҡ„еҗҢжӯҘж–№жі•пјҡйқһйҳ»еЎһ
  • дёӨдёӘзәҝзЁӢи®ҝй—®зҡ„жҳҜsynchronizedзҡ„йқҷжҖҒж–№жі•пјҡйҳ»еЎһ
  • еҗҢж—¶и®ҝй—®еҗҢжӯҘж–№жі•е’ҢйқһеҗҢжӯҘж–№жі•пјҡйқһйҳ»еЎһ
  • и®ҝй—®еҗҢдёҖеҜ№иұЎзҡ„дёҚеҗҢзҡ„жҷ®йҖҡеҗҢжӯҘж–№жі•пјҡйҳ»еЎһ
  • еҗҢж—¶и®ҝй—®йқҷжҖҒsynchronizedе’ҢйқһйқҷжҖҒsynchronizedж–№жі•пјҡйқһйҳ»еЎһ
  • ж–№жі•жҠӣеҮәејӮеёёеҗҺпјҢдјҡйҮҠж”ҫй”Ғ
