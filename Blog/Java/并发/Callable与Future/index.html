CallableдёҺFuture

JDKеҸӘжҸҗдҫӣдәҶдёӨз§ҚзәҝзЁӢеҗҜеҠЁж–№ејҸпјҢиҝҷдёӨз§Қж–№ејҸдёӯзҡ„run()ж–№жі•зҡ„иҝ”еӣһеҖјжҳҜvoidзұ»еһӢпјҢCallableдёҚз®—жҳҜзәҝзЁӢеҗҜеҠЁж–№ејҸпјҢThreadзұ»д№ҹе№¶жІЎжңүжҺҘж”¶CallableеҸӮж•°зҡ„жһ„йҖ ж–№жі•пјҢеҸӘжҺҘж”¶RunnableжҺҘеҸЈеҸӮж•°зҡ„жһ„йҖ ж–№жі•пјҢиӢҘиҰҒе°Ҷз»“жһңеҖјиҝ”еӣһпјҢйңҖиҰҒз”ЁеҲ°дёҖдёӘеҢ…иЈ…зұ»FutrueTaskе°ҶCallableеҢ…иЈ…жҲҗRunnableпјҢз„¶еҗҺдј йҖ’з»ҷThreadзҡ„жһ„йҖ ж–№жі•еҚіеҸҜпјҡ

еҜ№дәҺзәҝзЁӢжұ жқҘиҜҙпјҢзәҝзЁӢжұ execute(Runnable command)жҺҘеҸЈжҳҜж— иҝ”еӣһеҖјзҡ„пјҢдёҺд№ӢзӣёеҜ№еә”зҡ„жҳҜдёҖдёӘжңүиҝ”еӣһеҖјзҡ„жҺҘеҸЈFuture<T> submit(Callable<T> task)пјҢиҜҘж–№жі•е№¶дёҚжҳҜеңЁThreadPoolExecutorдёӯзӣҙжҺҘе®һзҺ°зҡ„пјҢиҖҢжҳҜеңЁе…¶зҲ¶зұ»AbstractExecutorServiceдёӯе®һзҺ°зҡ„гҖӮ

Callableе…¶е®һжҳҜз”ЁRunnableе®һзҺ°зҡ„пјҢеңЁsubmitеҶ…йғЁжҠҠCallableйҖҡиҝҮFutureTaskиҝҷдёӘAdapterиҪ¬жҚўжҲҗRunnableпјҢз„¶еҗҺйҖҡиҝҮexecuteжү§иЎҢгҖӮFutureTaskжҳҜдёҖдёӘAdapterеҜ№иұЎпјҢеҚіе®һзҺ°дәҶRunnableжҺҘеҸЈд№ҹе®һзҺ°дәҶFutureжҺҘеҸЈпјҢе…¶еҶ…йғЁеҢ…еҗ«дәҶдёҖдёӘCallableеҜ№иұЎпјҢд»ҺиҖҢе®һзҺ°дәҶCallableиҪ¬жҚўжҲҗRunnableгҖӮ

FutureTaskд»»еҠЎеҲқе§ӢиҝҗиЎҢзҠ¶жҖҒдёәNEWпјҢжҳҜжһ„йҖ еҮҪж•°дҝқиҜҒзҡ„пјҢINTERRUPTINGе’ҢINTERRUPTEDжҳҜд»»еҠЎдёӯй—ҙзҠ¶жҖҒпјҢе…¶дҪҷеӣӣз§ҚжҳҜд»»еҠЎз»ҲжӯўзҠ¶жҖҒпјҢд»»еҠЎзҡ„дёӯй—ҙзҠ¶жҖҒжҳҜдёҖдёӘзһ¬жҖҒпјҢйқһеёёзҹӯжҡӮгҖӮдё”д»»еҠЎдёӯй—ҙжҖҒе№¶дёҚд»ЈиЎЁд»»еҠЎжӯЈеңЁжү§иЎҢпјҢиҖҢжҳҜд»»еҠЎе·Іжү§иЎҢе®ҢпјҢжӯЈеңЁи®ҫзҪ®жңҖз»Ҳиҝ”еӣһз»“жһңпјҢж•…еҸӘиҰҒstateдёҚеӨ„дәҺNEWзҠ¶жҖҒпјҢе°ұиҜҙжҳҺд»»еҠЎе·Із»Ҹжү§иЎҢе®ҢжҜ•пјҢиҝҷйҮҢзҡ„жү§иЎҢе®ҢжҜ•жҳҜжҢҮдј е…Ҙзҡ„CallableеҜ№иұЎзҡ„callж–№жі•жү§иЎҢе®ҢжҜ•пјҢжҲ–иҖ…жҠӣеҮәдәҶејӮеёёпјҢиҝҗиЎҢзҠ¶жҖҒд»…еңЁsetгҖҒsetExceptionе’Ңcancelж–№жі•дёӯиҪ¬жҚўдёәз»ҲжӯўзҠ¶жҖҒгҖӮ

жүҖжңүзӯүеҫ…д»»еҠЎжү§иЎҢе®ҢжҜ•зҡ„зәҝзЁӢзҡ„йӣҶеҗҲжҳҜйҖҡиҝҮеҚ•еҗ‘й“ҫиЎЁе®һзҺ°зҡ„йҳҹеҲ—пјҢFutureTaskдёӯе°ҶWaitNodeеҚ•еҗ‘й“ҫиЎЁеҪ“еҒҡж ҲжқҘдҪҝз”ЁпјҢзЎ®еҲҮжқҘиҜҙжҳҜеҪ“еҒҡTreiberж ҲжқҘдҪҝз”Ёзҡ„пјҢдҪҝз”ЁCASжқҘе®ҢжҲҗе…Ҙж ҲеҮәж Ҳж“ҚдҪңгҖӮ

еҗҢж—¶еҸҜиғҪжңүеӨҡдёӘзәҝзЁӢйғҪеңЁиҺ·еҸ–д»»еҠЎзҡ„жү§иЎҢз»“жһңпјҢиӢҘд»»еҠЎиҝҳеңЁжү§иЎҢиҝҮзЁӢдёӯпјҢеҲҷе°ҶиҺ·еҸ–з»“жһңзҡ„зәҝзЁӢеҢ…иЈ…жҲҗWaitNodeжү”еҲ°Treiberж Ҳзҡ„ж ҲйЎ¶пјҢеҚіе®ҢжҲҗе…Ҙж Ҳж“ҚдҪңгҖӮз”ұдәҺFutureTaskдёӯзҡ„йҳҹеҲ—жң¬иҙЁдёҠжҳҜдёҖдёӘTreiberж ҲпјҢеҲҷдҪҝз”ЁиҜҘйҳҹеҲ—еҸӘйңҖиҰҒдёҖдёӘжҢҮеҗ‘ж ҲйЎ¶иҠӮзӮ№зҡ„жҢҮй’Ҳе°ұиЎҢдәҶпјҢеңЁFutureTaskдёӯwaitersеұһжҖ§еҚіжҳҜжҢҮеҗ‘ж ҲйЎ¶зҡ„жҢҮй’Ҳпјҡ

callableиЎЁзӨәиҰҒжү§иЎҢзҡ„д»»еҠЎжң¬иә«пјҢrunnerжҳҜжү§иЎҢcallableзҡ„зәҝзЁӢпјҢдёәдәҶдёӯж–ӯжҲ–еҸ–ж¶Ҳд»»еҠЎеҒҡеҮҶеӨҮпјҢеңЁиҝҗиЎҢж—¶жү§иЎҢrunж–№жі•ж—¶йҖҡиҝҮCASи®ҫзҪ®зҡ„гҖӮoutcomeжҳҜз”ЁжқҘеӯҳеӮЁCallableзҡ„жү§иЎҢз»“жһңжҲ–жҠӣеҮәзҡ„ејӮеёёгҖӮ

жү§иЎҢrunж–№жі•ж—¶пјҢдҪҝз”ЁCASж“ҚдҪңе°ҶrunnerеұһжҖ§и®ҫзҪ®дҪҚеҪ“еүҚзәҝзЁӢпјҢеҸҜи§ҒrunnerеұһжҖ§жҳҜеңЁиҝҗиЎҢж—¶иў«еҲқе§ӢеҢ–зҡ„гҖӮputOrderedIntе’ҢputIntVolatileжҳҜзӯүд»·зҡ„пјҢдҝқиҜҒдәҶstateзҠ¶жҖҒеҜ№е…¶д»–зәҝзЁӢзҡ„еҸҜи§ҒжҖ§гҖӮhandlePossibleCancellationInterruptж–№жі•иҰҒз»“еҗҲеҗҺйқўзҡ„cancelж–№жі•жқҘзңӢпјҢжЈҖжөӢеҸ‘зҺ°s = INTERRUPTINGпјҢиҜҙжҳҺcancelж–№жі•иҝҳжІЎжңүжү§иЎҢеҲ°дёӯж–ӯеҪ“еүҚзәҝзЁӢзҡ„ең°ж–№пјҢйӮЈе°ұзӯүеҫ…е®ғе°ҶstateзҠ¶жҖҒи®ҫзҪ®жҲҗINTERRUPTEDгҖӮ

йҰ–е…Ҳе°ҶwaitersеұһжҖ§зҡ„еҖјз”ұеҺҹеҖји®ҫзҪ®дёәnullпјҢwaitersеұһжҖ§жҢҮеҗ‘дәҶTreiberж Ҳзҡ„ж ҲйЎ¶иҠӮзӮ№пјҢе°ҶиҜҘеҖји®ҫдёәnullзҡ„зӣ®зҡ„е°ұжҳҜжё…з©әж•ҙдёӘж ҲгҖӮ

и®ҫзҪ®дёҚжҲҗеҠҹеҲҷifиҜӯеҸҘеқ—дёҚдјҡиў«жү§иЎҢпјҢиҝӣиЎҢдёӢдёҖиҪ®forеҫӘзҺҜпјҢиҖҢдёӢдёҖиҪ®forеҫӘзҺҜзҡ„еҲӨж–ӯжқЎд»¶еҸҲжҳҜwaiters!=null пјҢеҲҷиҜҙжҳҺиҜҘж–№жі•еҸӘжҳҜдёәдәҶзЎ®дҝқwaitersеұһжҖ§иў«жҲҗеҠҹи®ҫзҪ®жҲҗnullгҖӮиӢҘи®ҫзҪ®жҲҗеҠҹпјҢеҶ…еұӮеҫӘзҺҜзҡ„дҪңз”ЁжҳҜйҒҚеҺҶй“ҫиЎЁдёӯжүҖжңүзӯүеҫ…зҡ„зәҝзЁӢпјҢе№¶е”ӨйҶ’д»–д»¬гҖӮ

д»»еҠЎзҡ„еҸ–ж¶ҲпјҢиӢҘд»»еҠЎе·Із»Ҹжү§иЎҢе®ҢжҲҗгҖҒд»»еҠЎе·Із»Ҹиў«еҸ–ж¶ҲиҝҮдәҶгҖҒд»»еҠЎеӣ дёәжҹҗз§ҚеҺҹеӣ дёҚиғҪиў«еҸ–ж¶Ҳcancelж“ҚдҪңдёҖе®ҡеӨұиҙҘгҖӮcancelж“ҚдҪңиҝ”еӣһtrueе№¶дёҚд»ЈиЎЁд»»еҠЎзңҹзҡ„иў«еҸ–ж¶ҲдәҶпјҢиҝҷеҸ–еҶідәҺеҸ‘иө·cancelж—¶пјҢд»»еҠЎжүҖеӨ„зҡ„зҠ¶жҖҒгҖӮ

иӢҘеҸ‘иө·cancelж—¶д»»еҠЎиҝҳжІЎжңүејҖе§ӢиҝҗиЎҢпјҢеҲҷйҡҸеҗҺд»»еҠЎе°ұдёҚдјҡиў«жү§иЎҢпјҢиӢҘеҸ‘иө·cancelж—¶д»»еҠЎе·Із»ҸеңЁиҝҗиЎҢдәҶпјҢиӢҘmayInterruptIfRunningдёәtrueпјҢеҲҷеҪ“еүҚеңЁжү§иЎҢзҡ„д»»еҠЎдјҡиў«дёӯж–ӯпјҢеҗҰеҲҷеҸҜд»Ҙе…Ғи®ёжӯЈеңЁжү§иЎҢзҡ„д»»еҠЎз»§з»ӯиҝҗиЎҢпјҢзӣҙеҲ°е®ғжү§иЎҢе®ҢгҖӮ

FutureTaskдёӯдјҡж¶үеҸҠеҲ°дёӨзұ»зәҝзЁӢпјҢдёҖзұ»жҳҜжү§иЎҢд»»еҠЎзҡ„зәҝзЁӢеҸӘжңүдёҖдёӘпјҢFutureTaskзҡ„runж–№жі•е°ұз”ұиҜҘзәҝзЁӢжқҘжү§иЎҢпјӣдёҖзұ»жҳҜиҺ·еҸ–д»»еҠЎжү§иЎҢз»“жһңзҡ„зәҝзЁӢеҸҜд»ҘжңүеӨҡдёӘпјҢиҝҷдәӣзәҝзЁӢеҸҜе№¶еҸ‘жү§иЎҢпјҢжҜҸдёҖдёӘзәҝзЁӢйғҪжҳҜзӢ¬з«Ӣзҡ„пјҢйғҪеҸҜд»Ҙи°ғз”Ёgetж–№жі•жқҘиҺ·еҸ–д»»еҠЎзҡ„жү§иЎҢз»“жһңгҖӮж•…FutureTaskдёӯдҪҝз”ЁCAS stateеҸҳйҮҸеҠ LockSupport.park/unparkжқҘе®һзҺ°йҳ»еЎһе”ӨйҶ’жңәеҲ¶гҖӮиӢҘд»»еҠЎиҝҳжІЎжңүжү§иЎҢе®ҢпјҢеҲҷиҝҷдәӣзәҝзЁӢе°ұйңҖиҰҒиҝӣе…ҘTreiberж ҲдёӯжҢӮиө·пјҢзӣҙеҲ°д»»еҠЎжү§иЎҢз»“жқҹпјҢжҲ–иҖ…зӯүеҫ…зҡ„зәҝзЁӢиҮӘиә«иў«дёӯж–ӯгҖӮ

дҪҝз”ЁawaitDoneж–№жі•зӯүеҫ…д»»еҠЎиҝӣе…Ҙз»ҲжӯўжҖҒпјҢawaitDoneзҡ„иҝ”еӣһеҖјжҳҜд»»еҠЎзҡ„зҠ¶жҖҒпјҢиҖҢдёҚжҳҜд»»еҠЎзҡ„з»“жһңпјҢжңҖз»Ҳзҡ„з»“жһңжҳҜйҖҡиҝҮreportж–№жі•ж №жҚ®awaitDoneж”ҫеӣһзҡ„зҠ¶жҖҒжқҘеҲӨж–ӯд»ҺиҖҢиҝ”еӣһе…·дҪ“зҡ„е“Қеә”з»“жһңгҖӮ
